# 贪吃蛇游戏 (Snake Game)

一个使用 Python 和 Pygame 开发的经典贪吃蛇游戏。

## 环境要求

- Python 3.6+
- Pygame 库

## 安装依赖

首先需要安装 Pygame：

```bash
pip install pygame
```

## 运行游戏

1. 进入游戏目录：
```bash
cd /home/cosmos/桌面/claude/snake_game
```

2. 运行游戏：
```bash
python3 snake.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ (上箭头) | 向上移动 |
| ↓ (下箭头) | 向下移动 |
| ← (左箭头) | 向左移动 |
| → (右箭头) | 向右移动 |
| Q | 游戏结束后退出 |
| C | 游戏结束后重新开始 |

## 游戏规则

- 使用方向键控制蛇的移动
- 吃到红色食物会增长蛇的长度并得分
- 撞到墙壁或自己的身体，游戏结束
- 游戏结束后按 C 键可以重新开始

## 技术特点

- 使用 Pygame 进行图形渲染
- 流畅的动画效果
- 实时得分显示
- 支持游戏结束后的重新开始

## 自定义设置

你可以在代码中修改以下参数来自定义游戏：

- `self.dis_width` - 窗口宽度（默认：600）
- `self.dis_height` - 窗口高度（默认：400）
- `self.snake_speed` - 游戏速度（默认：15）
- 颜色设置可修改 RGB 值
