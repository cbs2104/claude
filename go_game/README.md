# Go Game - 命令行围棋

一个不需要视觉棋盘的高强度心算围棋游戏，人机对弈。

## 功能特性

- 19路标准围棋棋盘
- 玩家执黑先行，电脑执白
- 完整的围棋规则实现：
  - 提子（吃子）
  - 气终计算
  - 打劫规则
  - 禁入点检测
- 简单的AI策略（评估气和位置价值）
- 终局实地计算

## 运行方式

```bash
python go_game.py
```

## 操作说明

- 输入坐标落子，格式为 **列字母 + 行数字**（如 `Q16`、`T15`）
- 输入 `PASS` 停一手
- 输入 `END` 停一手并终局
- 游戏结束后自动计算胜负

## 示例

```
请输入坐标 (如 Q16), PASS(停一手), END(终局): A1
电脑落子于 Q16
```

## 依赖

- Python 3.6+
- 无需额外依赖包

## 规则

- 黑棋贴目 6.5 目
- 双方连续停手则终局
- 自动计算实地和提子数判定胜负
